## Development

### Install node

``` bash
nvm install 10.13.0 && nvm use 10.13.0
```

### Set environment variables

``` bash
AUTH_DOMAIN="<domain>"
AUTH_CALLBACK_URL="http://localhost:3000/auth"
AUTH_CLIENT_ID="<client_id>"
```

### Install dependencies

``` bash
npm install
```

### Start Dev Server

``` bash
npm run development
```
